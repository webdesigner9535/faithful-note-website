## Quick publish for free (GitHub Pages)

1. Create a new GitHub repo (or use your existing Faithful Note repo).
2. Upload this folder's contents (`index.html`, `style.css`, and `assets/`) to the repo root.
3. In GitHub: **Settings â†’ Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / folder: `/ (root)`
4. Save. Your site will publish at:
   `https://YOUR-USERNAME.github.io/REPO-NAME/`
